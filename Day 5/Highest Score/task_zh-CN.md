### 求和
Python有许多内置函数来帮助我们处理数字。其中一个可以帮助我们计算总和（总计）。
例如:
```
学生分数 = [180, 124, 165, 173, 189, 169, 146]
总分 = sum(学生分数)
```
但是`sum()`的背后效果是如何实现的呢？这段代码是由开发Python的人员编写的，它可能看起来像这样：

```
学生分数 = [180, 124, 165]

总和 = 0
对于分数在学生分数中:
    总和 += 分数

打印(总和)
```
